## üì¶ Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/SubhradeepBasu18/Portfolio2.git
   cd Portfolio2
   ```
2. **Install dependencies:**  
   If using npm:
   ```bash
   npm install
   ```
   Or if using yarn:
   ```bash
   yarn install
   ```

3. **Run locally:**  
   ```bash
   npm start
   ```
   or
   ```bash
   yarn start
   ```

4. **Visit:**  
   Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
